# BioASQ-12B-RAG data bundle

These JSONL files are a pinned local adaptation of
[`mattmorgis/bioasq-12b-rag`](https://huggingface.co/datasets/mattmorgis/bioasq-12b-rag)
at revision `6d6add1a6ec2090991386b5ae7608b71fd637bc4`.

The dataset is licensed separately from the project source code under
[CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/). The files were converted
to JSON Lines and frozen into the project-specific `dev` and `eval` layout; they are distributed
for non-commercial research under the same license. No official BioASQ leaderboard equivalence
is claimed.

File counts and SHA-256 digests are recorded in `data/manifests/bioasq.json`. The large JSONL
files are stored with Git LFS; run `git lfs pull` after cloning when needed.
